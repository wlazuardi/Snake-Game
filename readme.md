# Snake Game

Minimal classic Snake implemented with plain HTML, CSS, and JavaScript.

## Run

Open [index.html](/D:/codex/snake-game/index.html) in a browser, or serve the folder with any static file server and navigate to the root page.

## Manual checklist

- Start moving with arrow keys or `WASD`.
- Confirm the snake grows and score increments after eating food.
- Confirm `Pause` or `Space` stops movement and resumes cleanly.
- Confirm hitting a wall or the snake body ends the game.
- Confirm `Restart` resets score, snake position, and food placement.
